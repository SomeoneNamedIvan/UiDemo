# Ui Demo
Avanade Ui Demo

## NodeJs
Download and install nodejs version 11 and above
https://nodejs.org/en/

## Client ( /client )
### Install modules
```
npm install
```
or
```
yarn
```

To start development, do ( running on port 9090 )
```
npm run dev
```
or
```
yarn dev
```

## Server  ( /server )
### Install modules
```
npm install
```
or
```
yarn
```

start dev server ( running on port 9005 )
```
npm run nodemon
```
or
```
yarn nodemon
```
